## Quick Reference

### Day 1 Priorities

1. Get credentials (VPN, email, Teams, ticketing)
2. Register vendor accounts (Cisco CCO, F5, Juniper, Aruba)
3. Set up desktop (Putty, bookmarks, Outlook signature/rules)
4. Review documentation with support team

### Cardinal Rules

- **No unauthorized changes** on production network devices
- **Always have change control** before modifying devices
- **Always have a backout plan** for every change
